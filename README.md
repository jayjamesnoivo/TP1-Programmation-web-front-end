# TP1 – Programmation Web Front-End

## 📌 Description

Ce projet a été réalisé le 24/08/2025 dans le cadre du **TP1** du cours de programmation web front-end.

L’objectif est de créer un site web simple sur le thème du **système solaire**, en utilisant HTML et CSS, avec une structure organisée et une navigation entre plusieurs pages.

---

## 🎯 Objectifs

* Créer une structure HTML claire et valide
* Appliquer du style avec CSS
* Organiser les fichiers (images, pages, styles)
* Créer une navigation entre les pages
* Respecter les consignes du travail pratique

---

## 🛠️ Technologies utilisées

* HTML5
* CSS3

---

## 📂 Structure du projet

```bash
TP1-Jeremy/
│
├── css/
│   └── style.css
│
├── images/
│   ├── logo.png
│   └── systeme-solaire.jpg
│
├── pages/
│   ├── contact.html
│   └── planetes.html
│
└── index.html
```

---

## 🚀 Fonctionnalités

* Page d’accueil (`index.html`)
* Page des planètes (`planetes.html`)
* Page de contact (`contact.html`)
* Navigation entre les pages
* Intégration d’images
* Mise en page avec CSS

---

## ▶️ Comment exécuter le projet

1. Télécharger ou cloner le projet
2. Ouvrir le dossier dans Visual Studio Code
3. Ouvrir le fichier `index.html`
4. Lancer avec **Live Server** (recommandé)
